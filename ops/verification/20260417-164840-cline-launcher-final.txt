## Exact CLI command pattern used by the launcher

The launcher detects the Cline CLI via $CLINE_CLI (env override) or the
'cline' binary in PATH and invokes:

  cline ${CLINE_RUN_ARGS:-run} <prompt-file>

Default $CLINE_RUN_ARGS is 'run' (subcommand); override per-session if a
different CLI wrapper expects a different flag (e.g. --prompt-file).
Wall-clock timeout is applied via timeout(1) / gtimeout(1) when present.
Output is teed to ops/verification/YYYYMMDD-HHMMSS-cline-run-<stem>.log.

## How to queue a run_cline_prompt task (for future agents)

1. Commit a prompt file at .cursor/prompts/cline-prompt-<topic>.md.
2. Build task JSON at ops/work_queue/pending/<stamp>-<topic>.json with:
     task_type: run_cline_prompt
     payload: {prompt_file: '<repo-relative>', dry_run: false, timeout: 1800}
3. Sign with: /opt/homebrew/bin/python3 scripts/task_signer.py sign \
     --task <path> --priv ~/.config/symphony/<signer>.ed25519.priv
4. Commit + push. The launchd-scheduled runner picks it up within ~2 min.
5. Pull; read:
     - ops/verification/<task_id>-result.txt (runner log)
     - ops/verification/YYYYMMDD-HHMMSS-cline-run-<stem>.log (launcher log)

Full recipe: ops/work_queue/TASK_SCHEMA.md → 'How to queue a
run_cline_prompt task'.

## Limitations / TODOs

- Cline CLI is not installed on Bob today ('cline' not in PATH; no
  @cline/cli npm package exists as of this run). Live mode will exit 4
  until Matt (or another agent) installs a Cline CLI and either adds it
  to PATH as 'cline' or exports CLINE_CLI=/abs/path before invoking the
  runner. The launcher logs this as a blocker with clear resolution text
  rather than crashing the pipeline.
- The launcher invokes the CLI as '<cli> run <prompt-file>' by default.
  If the installed Cline CLI uses a different flag (--prompt-file, -f),
  set CLINE_RUN_ARGS accordingly (e.g. CLINE_RUN_ARGS='--prompt-file').
  A per-host override file can be added if Bob's CLI wrapper diverges.
- No automatic queueing of a follow-up 'verify cline-ran' task yet — if
  desired, wire a run_cline_prompt task and a follow-up verify_dump into
  a run_cline_campaign payload.
