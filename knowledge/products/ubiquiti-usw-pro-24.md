# Ubiquiti USW-PRO-24 24-Port Gigabit Switch

## Overview
A 24-port managed Layer 3 gigabit switch with 2x 10G SFP+ uplinks. Provides VLAN-aware switching for non-powered network devices throughout the home.

## Specs
- Ports: 24x 1GbE RJ45
- Uplinks: 2x 10G SFP+
- Layer 3: Static routing, inter-VLAN routing
- VLAN: 802.1Q, up to 4096 VLANs
- Management: UniFi Network (adopted by UDM-SE)
- Rack Space: 1U
- PoE: None

## Symphony Usage
Used alongside the PoE switch to provide port density for non-powered devices — streaming boxes, Control4 controllers, amplifiers, and other rack equipment. Rack-mounted and managed through the UDM-SE's UniFi Network application.
