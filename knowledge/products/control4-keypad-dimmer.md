# Control4 Contemporary 120V Keypad Dimmer

## Overview
A combination keypad and dimmer in one device. Provides configurable buttons for scene control while also dimming a connected lighting load. Part of Control4's native lighting platform, communicating directly with the CORE3 over ZigBee.

## Specs
- Load Rating: 120V, up to 600W incandescent / 150W LED
- Buttons: 6 configurable buttons with customizable LED colors
- Communication: ZigBee mesh
- Style: Contemporary series
- Color: Snow White (SW)
- Requires: Neutral wire

## Symphony Usage
Used at primary room entry points and high-traffic locations where scene control and dimming are both needed. Buttons are programmed for room scenes (e.g., "All On," "Relax," "Off"), audio control, or whole-home functions. LED feedback provides at-a-glance system status.
