# Triad TS-AMS16 Audio Matrix Switch

## Overview
A 16x16 analog audio matrix switch that routes any audio source to any output zone independently. Sits between the source devices (CA-1, TV audio returns, etc.) and the amplifiers.

## Specs
- Matrix: 16 inputs x 16 outputs (analog stereo)
- Switching: Non-blocking, any-to-any
- Control: RS-232, IP
- Rack Space: 1U
- Connectors: Phoenix/Euroblock

## Symphony Usage
Rack-mounted between audio sources and amplifiers. Enables flexible source routing — any room can listen to any source independently. Controlled via Control4 for seamless zone/source selection from keypads, touchscreens, or the app.
