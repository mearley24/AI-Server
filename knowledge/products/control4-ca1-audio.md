# Control4 CA-1 Audio

## Overview
The CA-1 is Control4's dedicated audio streaming endpoint. It provides high-resolution audio output from streaming services (Spotify, Tidal, Apple Music, etc.) into the distributed audio system via the amplifier chain.

## Specs
- Audio Output: Analog stereo (RCA), digital coax, digital optical
- Streaming: AirPlay, native Control4 streaming services
- Resolution: Up to 24-bit/192kHz
- Network: Ethernet (recommended), WiFi
- Power: 12V DC

## Symphony Usage
Rack-mounted as the primary audio source feeding the Triad amplifier system. Provides all streaming audio to the distributed audio zones. Typically one CA-1 per project unless multiple independent source pools are required.
