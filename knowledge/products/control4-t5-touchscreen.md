# Control4 T5 In-Wall Touchscreen 8" Midnight Black

## Overview
An 8-inch in-wall touchscreen running the Control4 interface natively. Provides dedicated system control at key locations — lighting scenes, audio, intercom, cameras, and schedules.

## Specs
- Display: 8" capacitive touchscreen, 1280x800
- Connectivity: WiFi, Ethernet (PoE optional)
- Intercom: Built-in camera, microphone, speaker
- Mounting: In-wall flush mount, requires 2-gang box + backing
- Color: Midnight Black (MB)

## Symphony Usage
**Being phased out in favor of iPad wall mounts on new projects.** iPads offer a better user experience, easier updates, and lower total cost when factoring in flexibility. T5 touchscreens are still used on legacy/existing projects or when a client specifically requests the dedicated Control4 interface. On the Topletz project, T5 units were removed and replaced with iPad mounts at Kitchen/TV Room and Master Bedroom per client direction (3/30/2026).
