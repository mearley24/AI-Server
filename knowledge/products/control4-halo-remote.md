# Control4 Halo Remote

## Overview
The Halo Remote is Control4's premium handheld remote with a color touchscreen, voice control, and customizable button layout. It provides full system control from the palm of the hand — lighting scenes, audio, video, and climate.

## Specs
- Display: 2.4" color touchscreen
- Connectivity: WiFi, ZigBee
- Voice: Built-in microphone for voice commands
- Charging: Inductive charging cradle included
- Battery: Rechargeable lithium-ion
- Color: Black (BL)

## Symphony Usage
Offered as a premium control option for main living areas and master bedrooms. Typically one per primary entertainment zone. Provides quick access to scenes, audio sources, and system functions without reaching for a phone or touchscreen.
