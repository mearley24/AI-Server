# Ubiquiti USW-PRO-24-POE 24-Port PoE++ Switch

## Overview
A 24-port managed gigabit switch with 802.3bt PoE++ (400W total budget) and 2x 10G SFP+ uplinks. Powers network devices that require PoE — access points, cameras, touchscreens, and intercoms.

## Specs
- Ports: 24x 1GbE RJ45 with PoE++
- PoE Budget: 400W total (802.3at/bt)
- Uplinks: 2x 10G SFP+
- Layer 3: Static routing
- VLAN: 802.1Q
- Management: UniFi Network
- Rack Space: 1U

## Symphony Usage
Powers all PoE devices on the network — WiFi access points, security cameras (when installed), and any PoE-powered endpoints. Rack-mounted alongside the non-PoE switch. Essential for camera and access point deployment.
