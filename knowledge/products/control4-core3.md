# Control4 CORE3 Controller

## Overview
The CORE3 is Control4's centralized automation controller that serves as the brain of the smart home system. It manages all integrated subsystems including lighting, audio, security, and user interfaces. Symphony uses the CORE3 as the standard controller on all residential projects.

## Specs
- Processor: Quad-core ARM
- OS: Control4 OS 3.x
- Connections: Ethernet, USB, ZigBee, Z-Wave, IR, Serial, Relay
- Audio Outputs: Analog stereo, digital coax/optical
- HDMI: 1 output
- Rack mountable: Yes (1U with optional ears)
- Power: 12V DC

## Symphony Usage
Installed in the equipment rack as the central controller for every Control4-based project. Manages all device communication, automation programming, scene logic, and user interface delivery. One CORE3 per residence is standard — CORE5 is reserved for large-scale or commercial projects.
