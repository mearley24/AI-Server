Install and configure one Control4 CORE3 controller as the centralized control platform for the residence.

Provide a fully integrated Control4 lighting control system including dimmers, switches, and keypads aligned with the approved load schedule and lighting design intent. Control4 native lighting was selected for this project per client direction.

Integrate lighting, distributed audio, security panel status, and user interfaces into a unified and intuitive control experience throughout the home.

Programming will include room naming and system organization. Lighting scenes will be tailored to each space. Keypad button assignments and user interaction design will be configured. Audio source routing and core automation functionality will be implemented.

Programming includes standard system functionality aligned with the project scope. Additional customization, revisions, or post-approval changes will be addressed as billable service.
