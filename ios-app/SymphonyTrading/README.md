# Symphony Trading - iOS App

Dedicated iOS app for trading workflows (Clawdbot/Polymarket/Crypto) using the Trading API process.

## Scope

- Portfolio and goal tracking
- Market scan + research actions
- Trading memory status from Cortex trading scope

## API

- Default base URL: `http://bobs-mac-mini:8421`
- Configure Trading API process in `api/trading_api.py`

## Build

From repo root:

```bash
bash ios-app/build_trading_simulator.sh
```

Open project:

```bash
open ios-app/SymphonyTrading/SymphonyTrading.xcodeproj
```
