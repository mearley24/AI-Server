Ollama not responding
brew services restart ollama — if still failing, check ~/.ollama/logs and ensure port 11434 is free.
Docker containers won't start
Run docker system prune -f to free disk, then docker-compose up -d --force-recreate.
Node unreachable from Bob
Verify SSH keys with ssh -v user@<NODE_IP>. Check firewall: sudo pfctl -sr on macOS.
Voice receptionist calls failing
Check Twilio webhook URL in console. Verify Docker: docker logs voice_receptionist. Ensure port 5000 is exposed.
HARPA extension not connecting
Re-enable in Chrome extensions. Clear extension cache. Verify API key in HARPA settings matches .env file.
